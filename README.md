# Student Performance Analysis

## Project Overview
This project analyzes student academic performance using Python, Pandas, Matplotlib, and Seaborn.

The objective is to explore the factors affecting students' final grades and gain insights through data visualization and statistical analysis.

## Dataset
The dataset used in this project is:
- student-mat.csv

It contains information about students including:
- Study Time
- Absences
- Family Support
- Previous Grades (G1, G2)
- Final Grade (G3)
- Demographic Information

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Tasks Performed
### Data Cleaning
- Loaded dataset
- Checked dataset structure
- Verified missing values
- Generated descriptive statistics

### Exploratory Data Analysis
- Distribution of Final Grades (G3)
- Study Time vs Final Grade Analysis
- Gender-wise Average Final Grade Comparison
- Grade Distribution Across Terms (G1, G2, G3)

## Visualizations
1. Histogram of Final Grades (G3)
2. Scatter Plot: Study Time vs Final Grade (G3)
3. Bar Chart: Average Final Grades by Gender
4. Box Plot: Grade Distribution Across G1, G2, and G3

## Key Insights
- Students with higher study time generally tend to achieve better grades.
- Grade distributions show variation among students.
- Gender-wise performance differences can be observed through average grades.
- Previous grades are useful indicators of final performance.
